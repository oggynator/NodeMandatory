What is Node.js?

Node.js is a runtime environment written in JavaScript that can be used for many development purposes, such as creating web apps or API's. Node.js is built on the same engine as Google Chrome, the V8 JavaScript engine, which performs very good, as it is both lightweight and efficient.

It uses a so called “non-blocking I/O model”, which enables Node.js to independently handle multiple input/output functions at the same time. This makes Node.js very good for web applications, as it enables it to render data faster, as it can handle multiple requests at the same time.

Node.js comes bundled with NPM, which is a package manager(the largest in the world). More info on NPM here.
