What is jQuery and how can it help us?

jQuery is a Javascript library with the motto: “Write Less Do More”.

It provides developers with faster, safer and more robust code. This is because it is quality tested and optimized and prevents developers from thinking too much about declaration and more about implementation.

Basic syntax of jQuery code is: $(selector).action()
