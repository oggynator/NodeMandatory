What is a REST API?

An API is a set of rules and functionality that software follows when it communicates with other software.

REST API (representational state transfer) is a software architectural style, that defines what a piece of software can provide how it is provided.

An API that is considered RESTful will often provide some if not all of the four core operations of CRUD(Create, Read, Update, Delete).

Implemented API

This web app has its own implented API which handles mobile devices, it supports all the CRUD functions.
The url to access the devices is:

localhost:3000/rest/devices

This API could be expanded and then used in another application or maybe to generate a webpage that sells or informs customers about mobile devices. The JSON format can be converted to HTML with a tempating library like json2html, which can be found in npm.
